CSS Situations:

How to center a div:

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center
}

OR

.container {
  display: grid;
  place-items: center
}


<div class="container">
   <div class="box">Centered</div>
</div>
```

How to make something occupy full space of parent:

```css
.parent {
width: 400px;
height: 300 px;
}

.child {
width: 100%;
height: 100%;
}

OR

.parent {
display: flex;
}

.child {
flex:1;
}

OR

.parent {
display: grid;
}

.child {
width: 100%;
height: 100%;
}

OR

.parent {
position: relative;
}

.child {
position: absolute;
inset:0;
}

<div class="parent">
   <div class="child"></div>
</div>
```

Absolute & relative positioning:

```css
.card {
position: relative;
width: 200px;
height: 120px;
background: #eee;
}

.badge {
position: absolute;
top: 10px;
right: 10px;
background: crimson;
color: white;
padding: 4px 8px;
font-size: 12px;
border-radius: 12px;
}

<div class="card">
  <span class="badge">NEW</span>
  <h3>Product Title</h3>
  <p>This is a simple card example.</p>
</div>

```
