Interacting with a remote VM:

1. SSH into the VM: ssh username@vm-ip-address
2. SSH using a private key: ssh -i path/to/private-key.pem username@vm-ip-address
3. Change password: passwd
4. Switch to another user: su username
5. Switch to root user: sudo su
6. Copy file from local machine to VM: scp file.txt username@vm-ip-address:/path/on/vm
7. Copy file from VM to local machine: scp username@vm-ip-address:/path/on/vm/file.txt .
8. Copy directory recursively to VM: scp -r folder username@vm-ip-address:/path/on/vm
9. Restart the VM: sudo reboot
10. Shutdown the VM: sudo shutdown now
11. Find process running on a port (6000): sudo netstat -plten | grep :60000
12. Kill process on a port: kill `<PID>` (Process ID is shown in last column alongwith program name)
13. Windows - find process running on port (3000): netstat -ano | findstr :3000
14. Windows - forcefully terminate a process: taskkill /PID <Process_ID> /F
15. Copy file from VM1 to VM2 : scp file.txt username@vm2-ip:/path/on/vm2/
